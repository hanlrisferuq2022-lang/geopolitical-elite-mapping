# 置信度规范 (Confidence Specification)

> ⛔ **核心原则**：地缘政治数据不透明、存在叙事竞争、没有 ground truth。置信度标注是诚实性的核心保障，不是可选注释。

## 四级置信度系统

| 等级 | 缩写 | 判定标准 | 最低证据要求 |
|------|------|---------|------------|
| **HIGH** | `[H]` | 事实明确、多源交叉验证通过 | 3+ 独立来源，含至少 1 个官方文件或权威学术来源 |
| **MED** | `[M]` | 较可信但存在信息缺口 | 2 个独立来源，或 1 个权威来源 + 合理推断链 |
| **LOW** | `[L]` | 信息有限或来源有偏见 | 仅 1 个来源，或多来源但均有偏见倾向 |
| **INFERRED** | `[I]` | 无直接证据，从行为模式推断 | 基于已知事实的逻辑推断，必须标注推断链 |

## 判定流程

### 节点置信度判定

```
收集到某行为体的信息
  ├── 有 3+ 独立来源描述其角色和影响力？
  │   ├── YES → 来源中含官方文件 OR 权威学术来源？
  │   │   ├── YES → ✅ HIGH
  │   │   └── NO → 来源间一致性高？
  │   │       ├── YES → ✅ HIGH
  │   │       └── NO → ⚠️ MED + 标注分歧点
  │   └── NO → 有 2 个独立来源？
  │       ├── YES → 来源质量如何？
  │       │   ├── 至少 1 个权威来源 → ⚠️ MED
  │       │   └── 均为新闻/二手来源 → ⚠️ LOW
  │       └── NO → 仅 1 个来源？
  │           ├── YES → ⚠️ LOW + 标注来源局限
  │           └── NO（0 来源） → 从行为推断？
  │               ├── YES + 逻辑链完整 → ❓ INFERRED
  │               └── NO → ❌ 不纳入图谱
```

### 边（关系）置信度判定

```
声称 A 与 B 存在某种关系
  ├── 有官方文件/条约文本确认？
  │   └── YES → ✅ HIGH
  ├── 有 2+ 独立来源报道？
  │   ├── YES → 来源间存在叙事竞争？
  │   │   ├── YES → ⚠️ MED + 标注两方叙事
  │   │   └── NO → 来源含权威来源？
  │   │       ├── YES → ✅ HIGH
  │   │       └── NO → ⚠️ MED
  │   └── NO → 仅 1 个来源？
  │       ├── YES → 来源是冲突一方？
  │       │   ├── YES → ⚠️ LOW + 标注偏见风险
  │       │   └── NO → ⚠️ LOW
  │       └── NO → 从公开行为模式推断？
  │           ├── YES + 推断链合理 → ❓ INFERRED + 标注推断链
  │           └── NO → ❌ 不纳入图谱
```

## 来源质量分层

| 来源类型 | 质量等级 | 说明 |
|---------|---------|------|
| 官方文件（条约文本、政府公报、UN 决议） | ⭐⭐⭐ 权威 | 最高可信度，但注意官方叙事也可能有偏 |
| 学术研究（同行评审期刊、智库报告） | ⭐⭐⭐ 权威 | 高可信度，但注意智库的资金来源和立场 |
| 权威新闻（路透社、AP、BBC、FT 等） | ⭐⭐ 可靠 | 较高可信度，注意报道时效和地区偏见 |
| 泄露文件（WikiLeaks、Panama Papers 等） | ⭐⭐ 可靠 | 文件本身可信但需验证完整性和语境 |
| 地区新闻/行业媒体 | ⭐ 有参考价值 | 需交叉验证，可能有地区性偏见 |
| 社交媒体/自媒体/匿名来源 | ⚠️ 低可信度 | 仅作为线索，不能作为独立证据 |
| 冲突一方的官方声明 | ⚠️ 有偏见 | 有信息价值但必须有对方来源交叉验证 |

## 叙事竞争处理规则

> ⚠️ 地缘政治领域几乎所有重大事件都存在叙事竞争。以下规则确保图谱保持中立。

1. **识别叙事竞争**：当同一关系/事件存在 2+ 种互相矛盾的描述时，标记为叙事竞争
2. **不选边站**：图谱中不采用任何一方的叙事作为"事实"
3. **标注方式**：
   - 边 label 使用中性描述（如 "军事冲突" 而非 "侵略" 或 "解放"）
   - 报告中并列展示各方叙事
   - 置信度反映的是"关系存在"的确信度，不是某方叙事的正确性
4. **特殊标签**：存在叙事竞争的边，在报告验证表中标注 `🔀 叙事竞争`

## 置信度分布健康度指标

Phase 3 完成后，必须统计并报告：

| 指标 | 健康范围 | 异常处理 |
|------|---------|---------|
| HIGH 占比 | 20-50% | <20% = 数据基础薄弱，需在报告中特别说明 |
| MED 占比 | 30-50% | 主体部分，正常 |
| LOW 占比 | 10-30% | >30% = 考虑削减低质量节点/边 |
| INFERRED 占比 | 0-15% | >15% = 推断过多，需审视推断链质量 |
| 无标注占比 | ⛔ 0% | >0% = 禁止进入 Phase 4 |
