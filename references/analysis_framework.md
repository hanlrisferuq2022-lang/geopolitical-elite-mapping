# 地缘政治分析框架

> 本框架替代 technology-mapping 中的 VC 分析框架。图谱的最终用户是**地缘政治研究者和决策者**，图谱必须回答以下问题。

## A. 权力结构分析

- 哪些行为体是**权力中心**（拥有最多高置信度连接的节点）？
- 权力的来源是什么（军事力量、经济资源、制度权力、意识形态影响力）？
- 权力是**集中的**（少数节点控制大部分连接）还是**分散的**（多极均衡）？
- 在图谱上的表现：权力中心 = 连接数最多 + 粗边框节点；权力结构 = cluster 大小和 cluster 间边密度

## B. 利益冲突热点

- 哪些行为体之间存在**结构性矛盾**（红色边集中区域）？
- 冲突的根源是什么（资源争夺、安全困境、意识形态对立、历史遗留）？
- 是否存在**潜在爆发点**（置信度低但战略重要的冲突关系）？
- 在图谱上的表现：红色边密集区 = 冲突热点；INFERRED 红色边 = 潜在爆发点

## C. 关键脆弱节点

- 移除哪个节点会导致**网络重组**（连接两个或更多 cluster 的桥梁节点）？
- 哪个行为体的"退出"或"转向"会最大程度改变现有权力平衡？
- 在图谱上的表现：标注 `🔗 关键连接点`，通常位于两个 cluster 之间

## D. 叙事竞争图景

- 同一议题有哪些**竞争性叙事**（如"自由秩序"vs"多极化"）？
- 哪些行为体是各叙事的**主要推动者**？
- 叙事如何影响公开可得的数据质量和偏见方向？
- 在图谱上的表现：存在叙事竞争的边/节点标注 `🔀`

## E. 动态趋势分析

- 近 5-10 年哪些关系**显著增强或减弱**？
- 是否有**新的行为体**正在崛起（近年新出现的节点）？
- 是否有**传统行为体**在衰退（连接减少/断裂的节点）？
- 在图谱上的表现：时间轴锚定 + 边 label 中的年份变化

---

## 分析标注（每个核心行为体节点必须包含）

| 🏷️ 标签 | 含义 | 判据 |
|---------|------|------|
| `🏷️ 权力中心` | 该行为体是当前议题的权力中心 | 3+ 条 HIGH 置信度主边连接 |
| `🏷️ 桥梁节点` | 连接两个或更多对立阵营 | 跨 cluster 边 ≥ 2 条 |
| `🏷️ 依赖方` | 在关键资源/安全上依赖他方 | 有 ≥ 1 条资源控制/安全同盟的从属边 |
| `🏷️ 争议行为体` | 该行为体的角色存在叙事竞争 | 相关边有 `🔀 叙事竞争` 标注 |
| `🏷️ 新兴力量` | 近 10 年内新出现或显著崛起 | 时间轴锚定在近期 + 连接增长 |
| `🏷️ 衰退力量` | 近 10 年影响力显著下降 | 历史上有更多连接，但近期连接断裂 |

## 报告分析章节必须包含

### 1. 权力拓扑总结
- 图谱的整体结构是什么类型？（两极对抗 / 多极均衡 / 单极主导 / 碎片化）
- 最核心的 3-5 个行为体及其权力来源

### 2. 冲突轴线图
- 主要冲突轴线（如 A 阵营 vs B 阵营）
- 各轴线的冲突维度（经济 / 军事 / 意识形态 / 资源）

### 3. 脆弱性评估
- 当前网络的关键脆弱点
- 最可能导致网络重组的scenario（假设 X 退出/崩溃，会怎样？）

### 4. 信息可靠性评估
- 哪些区域的数据最可靠？哪些最不可靠？
- INFERRED 关系的汇总及其推断依据
- 叙事竞争对分析的影响
